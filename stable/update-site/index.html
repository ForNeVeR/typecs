Installation

  1. Required software: Windows 7, Java 7, Eclipse Kepler 4.3 or higher, Node.js.
  2. In Eclipse go to Help > Install New Software...
  3. Provide the installation location http://axmor.github.io/typecs/stable/update-site/
  4. Mark the plugin version you would like to install then press Next...
  5. On Install Details press Next...
  6. Review and confirm the plugin to install.
  7. Restart Eclipse.
